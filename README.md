# PlainTalk

PlainTalk is a simple web app I made for the Dementia Hackathon.

The idea was to make technology feel less overwhelming for dementia patients by simplifying information and keeping everything visual and easy to use.

There are two parts to the app:
- A patient view with large buttons, reminders, reading, music and photos
- A carer dashboard where reminders, stories and other content can be managed

One feature I focused on was simplifying difficult news articles into shorter and clearer sentences using the OpenAI API.

## Running the project

Just open `index.html` in a browser.

You can also use Live Server in VS Code if you want everything to update automatically while editing.

## Files

- `index.html` → patient interface
- `admin.html` → carer dashboard
- `css/` → styling
- `js/` → functionality

## Built With

- HTML
- CSS
- JavaScript
- OpenAI API
- localStorage

## Why I made it

A lot of apps for older people still feel cluttered or confusing. I wanted to experiment with something calmer, simpler and more supportive for people with dementia. and clear text.
